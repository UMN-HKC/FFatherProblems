## Solution1
``` java  
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> res = new ArrayList<>();
        if (nums == null || nums.length == 0) {
            return res;
        }
        backtrack(res, nums, 0, new ArrayList<>());
        return res;
    }
    private void backtrack(List<List<Integer>> res, int[] nums, int index, List<Integer> list) {
        if (index == nums.length) {
            res.add(new ArrayList(list));
            return;
        }
        backtrack(res, nums, index + 1, list);      // not add current digit
        list.add(nums[index]);                      // add current digit
        backtrack(res, nums, index + 1, list);      
        list.remove(list.size() - 1);               // backtrack
    }
}
```

## note
* Backtracking 
* time O(2^n) because for n elements, we have 2^n subsets / space O(n)
